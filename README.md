# MindfulDo - Obsidian Plugin

*MindfulDo • Do mindfully*

*A calm, beautiful and simple todo list for Obsidian that helps you stay organized without stress.*

*O listă de task-uri calmă, frumoasă și plină de funcționalități pentru Obsidian care te ajută să te organizezi fără stres.*

![Plugin Preview](https://img.shields.io/badge/Obsidian-Plugin-purple) ![License](https://img.shields.io/badge/License-MIT-green) ![Version](https://img.shields.io/badge/Version-1.0.0-blue)

## 🌟 Features / Funcționalități

### 📝 **Task Management / Gestionarea Task-urilor**
- **Categorized tasks** with 6 pre-defined categories (Work, Personal, Health, Learning, Hobby)
- **Real-time task counting** with completion tracking
- **Smooth animations** for a relaxing user experience
- **Local storage** - all data stays in your Obsidian vault

- **Task-uri categorizate** cu 6 categorii predefinite (Lucru, Personal, Sănătate, Învățare, Hobby)
- **Numărare în timp real** cu urmărirea completării
- **Animații fluide** pentru o experiență relaxantă
- **Stocare locală** - toate datele rămân în vault-ul tău Obsidian

### ⏰ **Smart Reminders / Reminder-e Inteligente**
- **Intuitive date/time picker** - separate inputs for better UX
- **Browser notifications** with custom icons
- **Flexible expiration handling**:
  - Keep expired reminders until manually deleted
  - Auto-delete reminders when they expire
- **Persistent across sessions** - reminders survive Obsidian restarts

- **Selector intuitiv pentru dată/oră** - input-uri separate pentru UX mai bun
- **Notificări în browser** cu icoane personalizate
- **Gestionare flexibilă a expirării**:
  - Păstrează reminder-ele expirate până la ștergerea manuală
  - Șterge automat reminder-ele când expiră
- **Persistente între sesiuni** - reminder-ele supraviețuiesc restart-urilor Obsidian

## 🚀 Installation / Instalare

### Method 1: Manual Installation / Metoda 1: Instalare Manuală

1. **Download the plugin files** / **Descarcă fișierele plugin-ului**:
   - `main.js`
   - `manifest.json`
   - `styles.css`

2. **Create plugin folder** / **Creează folderul plugin-ului**:
   ```
   YourVault/.obsidian/plugins/mindfuldo/
   ```

3. **Copy files** / **Copiază fișierele** into the plugin folder

4. **Enable in Obsidian** / **Activează în Obsidian**:
   - Go to Settings → Community Plugins
   - Find "MindfulDo" and toggle it on
   - Mergi la Settings → Community Plugins
   - Găsește "MindfulDo" și activează-l

### Method 2: BRAT (Beta) / Metoda 2: BRAT (Beta)

1. Install BRAT plugin if you haven't already
2. Add this repository URL in BRAT settings
3. Enable the plugin in Community Plugins

## 🎯 Usage / Utilizare

### Getting Started / Începerea
1. **Open the plugin** by clicking the checkmark icon in the left sidebar
2. **Set your preferences** in Settings → MindfulDo Settings
3. **Start adding tasks** with the intuitive interface

1. **Deschide plugin-ul** apăsând pe iconița checkmark din sidebar-ul stâng
2. **Setează preferințele** în Settings → MindfulDo Settings  
3. **Începe să adaugi task-uri** cu interfața intuitivă

### Adding Tasks / Adăugarea Task-urilor
- Type your task in the input field
- Press Enter or click "Add" / "Adaugă"
- Tasks automatically categorize based on your current filter
- Click the checkbox to mark as complete

- Tastează task-ul în câmpul de input
- Apasă Enter sau click pe "Add" / "Adaugă"
- Task-urile se categorizează automat pe baza filtrului curent
- Click pe checkbox pentru a marca ca finalizat

### Setting Reminders / Setarea Reminder-elor
- Enter reminder description
- Pick date from calendar picker
- Choose time with time picker
- Click "Add" / "Adaugă"

- Introdu descrierea reminder-ului
- Alege data din calendar
- Selectează ora cu selectorul de timp
- Click pe "Add" / "Adaugă"
